# EventsApp

***designed by Gabriel Conte***

### Prerequisites

In order to run this project you must have installed Xcode and Cocoapods.

```
$ gem install cocoapods
```
Through terminal go to the folder of your project and run the following command:

```
$ pod instal
```

**Highlights**
